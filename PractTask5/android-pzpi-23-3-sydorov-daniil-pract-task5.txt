 Завдання: 
 1. Ознайомитися з основними типами сенсорів у Android.
 2. Опанувати процес зчитування даних із сенсорів та їх обробку.
 3. Навчитися використовувати клас SensorManager для керування сенсорами.

Хід роботи:
 1. Створюємо застосунок для зчитування даних акселерометра.
 1.2 Реалізувати функцію для обробки змін положення пристрою
 2. Додати у застосунок роботу з гіроскопом
 2.1 Вивести значення на екран
 2.2 Додати візуальну індукцію, рух зображення відповідно повороту екрану
 3. Додати до застосунку можливість вибору сенсора для зчитування даних
 3.1 реаізувати механізм для відображення даних обраного сенсора.
